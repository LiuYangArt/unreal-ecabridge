# Clean-room Triage Labels

- `port`: native has a useful capability ECABridge should reimplement from a behavior spec.
- `mirror-api`: native schema or UX shape should influence ECABridge, but behavior stays ECABridge-specific.
- `ignore`: internal-only, not useful for launcher users, or covered better by existing ECABridge tools.

Do not store Epic source snippets in reports, docs, issues, or commits.
